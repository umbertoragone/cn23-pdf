The CN23 form is usually delayed for the same reasons: vague descriptions, mismatched totals, or missing references.

## Use specific item descriptions

Descriptions like `gift`, `sample`, or `accessories` are usually too broad. Write what the product actually is, for example `cotton t-shirt` or `printed circuit board`.

## Keep totals consistent

Check that:

- line item weights add up to the total weight
- line item values match the total declared value
- invoice references match the documents you include with the parcel

## Double-check destination data

A correct customs declaration still fails if the recipient address is incomplete. Verify postcode, city, country, and any required contact details before printing.

## Final review

Before downloading the PDF, scan the form once as if you were the customs officer reading it for the first time. Ambiguous fields are usually obvious on that last pass.
