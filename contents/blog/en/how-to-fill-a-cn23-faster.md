If you ship the same kinds of parcels repeatedly, the fastest workflow is to create a **pre-filled URL** and reuse it.

## How it works

Every time you edit the form, the page updates the query string in the address bar. That means the URL already contains the values currently visible in the form.

## Practical workflow

1. Fill the sender details you use most often.
2. Add the most common parcel line items.
3. Save that page URL in your bookmarks.
4. Open it again whenever you need a new CN23 and only change the fields that vary.

## When this is useful

- Small stores with recurring international shipments
- People sending similar parcels to the same country
- Teams that want a shared starting template without maintaining a separate system

You should still review the declared values, item descriptions, and customs codes before downloading the final PDF.
